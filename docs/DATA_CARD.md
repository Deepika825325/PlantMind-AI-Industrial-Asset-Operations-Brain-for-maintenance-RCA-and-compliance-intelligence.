# PlantMind Nexus Data Card

## Data Scope

PlantMind Nexus uses demo industrial data for:

- Assets
- Documents
- Compliance rules
- Work orders
- RCA evidence
- Telemetry
- Incidents
- Model registry metadata
- Evaluation reports

## Core Demo Assets

- P-101
- C-201
- HX-301

## Data Types

| Type | Example |
|---|---|
| Asset metadata | P-101, C-201, HX-301 |
| Documents | SOP, inspection report, incident report |
| Telemetry | Vibration, temperature, anomaly score |
| Work orders | Maintenance lifecycle records |
| Compliance | Rule status and missing evidence |
| Model registry | Model version, metrics, stage |

## Data Quality Notes

Demo data is deterministic and designed for repeatable product demonstration. Production use requires real plant data validation, access control, lineage, and retention policy.
