# PlantMind AI Frontend

## Purpose

This is the frontend UI for PlantMind AI.

It connects to the FastAPI backend and visualizes industrial asset intelligence, compliance gaps, maintenance events, document evidence, knowledge graph relationships, and Ask PlantMind responses.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- FastAPI backend integration
